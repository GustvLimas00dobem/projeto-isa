# Florisa Floricultura

Site de floricultura criado com React + Vite, com login decorativo, catálogo de flores e carrinho de compras simulado em memória.

## Como rodar

```bash
npm install
npm run dev
```

Depois, abra o endereço exibido no terminal, normalmente `http://localhost:5173`.

## Rotas

- `/` - Login simulado
- `/home` - Catálogo e carrinho

Não há backend, banco de dados, autenticação real ou integração de pagamento.
